1. Рушайте до Консолі розробника Google та приєднайте до вашого Google аккаунту
  2. Натисніть Створити проект та надайте йому ім’я
  3. Як тільки цей процес завершиться, натисніть на новому проєкті
  4. Перейдіть до APIS & AUTH / Повноваження
  5. Натисніть Створити новий ID клієнта
  6. Надайте ім’я вашого домену в полі Authorized Javascript origins та в полі вище Callback URL Authorized redirect URI
  7. Зараз скопіюйте ID Клієнта та Секрет Клиєнта в налаштування Google2Piwigo
  8. Ви впорались!
