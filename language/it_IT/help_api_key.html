1. Vai alla Google Developers Console e collega il tuo account Google
  2. Clicca su Crea progetto e dai un nome
  3. Una volta finito il processo, clicca sul progetto appena creato
  4. Vai a APIS & AUTH / Credenziali
  5. Clicca su Crea nuovo Client ID
  6. Inserisci il tuo nome di dominio come Authorized Javascript origins e il URL di ritorno sopra come reindirizza URI autorizzato
  7. Ora copia l'Client ID e il Client Secret nella configurazione Google2Piwigo
  8. Hai finito!
