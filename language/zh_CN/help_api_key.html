1. 前往 Google Developers Console 并使用 Google 帐号连接
  2. 点击 Create project 并为之命名
  3. 进程完成后，点击新建的项目
  4. 前往 APIS & AUTH / Credentials
  5. 点击 Create new Client ID
  6. 将您的域名设为 Authorized Javascript origins 并将上面的 Callback URL 设为 Authorized redirect URL
  7. 现在将 Client ID 和 Client Secret 拷贝至 Google2Piwigo 设置
  8. 完成！
