1. Ga naar Google Developers Console en verbind met je Google account
  2. Klik Create project en geef het een naam
  3. Eens het proces afgerond is, klik op het nieuw aangemaakte project
  4. Ga naar APIS & AUTH / Credentials
  5. Klik Create new Client ID
  6. Geef je domein een naam zoals Authorized Javascript origins en de bovenvermelde Callback URL als Authorized redirect URI
  7. Kopieer nu de Client ID en de Client Secret naar de Google2Piwigo configuratie
  8. Je bent klaar!
