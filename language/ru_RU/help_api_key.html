1. Перейдите в Google Developers Console и подключитесь с использованием Вашего аккаунта Google
  2. Кликните Создать проект и задайте имя
  3. После завершения процесса нажмите на вновь созданный проект
  4. Перейдите в APIS & AUTH / Учетные данные
  5. Кликните Создать новый ID клиента
  6. Укажите свое доменное имя в качестве Authorized Javascript origins и выше Ответный URL в качестве Authorized redirect URI
  7. Теперь скопируйте ID клиента и Client Secret в конфигурацию Google2Piwigo
  8. Всё готово!
