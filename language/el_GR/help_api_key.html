1. Πήγαινε στο Google Developers Console και συνδεθείτε με το Google account
  2. Κάντε κλικ στο Create project δώστε ένα όνομα.
  3. Μόλις ολοκληρωθεί η διαδικασία, κάντε κλικ στο νεοσυσταθέν έργο.
  4. Πήγαινε στο APIS & AUTH / Credentials
  5. Κάντε κλικ στο Create new Client ID
  6. Δώστε το όνομα του domain σας, ως Authorized Javascript origins και το παραπάνω Callback URL ως Authorized redirect URI
  7. Τώρα αντιγράψτε το Client ID και το Client Secret στο διαμόρφωση Google2Piwigo
  8. Τελειώσατε!
