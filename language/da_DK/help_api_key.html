1. Gå til Google Developers Console og log på med din Google-konto
  2. Klik på Create project og giv projektet et navn
  3. Når processen er afsluttet, klikkes der på det nyligt oprettede projekt
  4. Gå til APIS & AUTH / Credentials
  5. Klik på Create new Client ID
  6. Angiv dit domænenavn som Authorized Javascript origins og den ovenstående Callback URL som Authorized redirect URI
  7. Kopier nu Client ID og Client Secret til Google2Piwigo-opsætningen
  8. Færdig!
