1. Ir para Console de Desenvolvedores Google faça conexão usando sua conta do Google
  2. Click Criar projeto e dê um nome
  3. Uma vez que o processo concluído, clique no projeto recém-criado
  4. Ir para APIS & AUTH / Credenciais
  5. Click Criar novo Cliente ID
  6. Dê o seu nome de domínio como Origens Javascript Autorizados e o acima Chamar de volta a URL as URI redirecionamento Autorizado
  7. Agora copie o ID do Cliente e o Segredo do Cliente para a configuração do Google2Piwigo
  8. Está pronto!
