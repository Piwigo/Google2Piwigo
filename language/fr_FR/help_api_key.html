1. Allez sur Google Developers Console et connectez-vous avec votre compte Google
  2. Cliquez sur Create project et entrez un nom
  3. Dès que le processus de création est terminé, cliquez sur l'application nouvellement créée
  4. Rendez-vous dans APIS & AUTH / Credentials
  5. Cliquez sur Create new Client ID
  6. Entrez votre nom de domaine pour Authorized Javascript origins et le Callback URL ci-dessus pour Authorized redirect URI
  7. Maintenant copiez le Client ID et le Client Secret dans la configuration de Google2Piwigo
  8. Terminé!
