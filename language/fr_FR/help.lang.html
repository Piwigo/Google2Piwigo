1. Allez sur la page https://accounts.google.com/ManageDomains.
  2. Écrivez votre nom de domaine sans le sous-domaine dans le champs (ei: mydomain.com, non gallery.mydomain.com) et cliquez sur Add domain.
  3. Cliquez sur Manage [mydomain.com] et acceptez les conditions d'utilisation (ToS).
  4. Écrivez votre nom de domaine AVEC le sous-domaine (même si c'est www) et le préfixe http:// dans le premier champs, et validez.
