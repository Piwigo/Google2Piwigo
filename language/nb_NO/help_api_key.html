1. Gå til Google Developers Console og koble deg på med din Google-konto
  2. Klikk Opprett et prosjekt og gi det et navn
  3. Når prosessen er ferdig, klikker du på det nyopprettede prosjektet
  4. Gå til APIS & AUTH / legitimasjon
  5. Klikk Opprett ny bruker-ID
  6. Gi ditt domenenavn en godkjenning som Autorisert Javascript opprinnelse og det ovenfor Callback URL som enAutorisert omdirigert URL
  7. Nå kopier Bruker-ID og Bruker Legitimasjon til Google2Piwigo konfigurasjon
  8. Du er ferdig!
